# 妙阅（ai-bookreader）

基于 ESP32-S3 + 4.2 寸三色墨水屏的电子书阅读器

## 硬件规格

| 项目 | 规格 |
|------|------|
| 主控 | ESP32-S3（N8R2：8MB Flash，无 PSRAM） |
| 显示屏 | 4.2 寸三色墨水屏（SSD1683，400×300，黑白红） |
| 存储 | SD 卡（SPI）+ NVS |
| 按键 | 5 个（PWR/PREV/NEXT/HOME/自定义） |

## 功能特性

- TXT / EPUB 书籍阅读
- FreeType 矢量字体渲染
- 自动分页与阅读进度保存
- WiFi 配网与天气显示
- 网页配置界面
- 低功耗深度睡眠

## PC 仿真（SDL2）

项目支持使用 SDL2 在 PC 上运行仿真，方便开发调试。仿真使用 Mock 数据层模拟真实硬件数据。

### 前置依赖

- SDL2（`brew install sdl2`）
- FreeType（`brew install freetype`）

### 构建与运行

```bash
# 进入仿真目录
cd app

# 编译
make clean && make

# 运行仿真
./reader_sim
```

### 操作按键

| 按键 | 功能 |
|------|------|
| `ESC` | 退出仿真 |
| `←` 左箭头 | 上一个菜单 |
| `→` 右箭头 | 下一个菜单 |
| `Enter` | 确认/进入 |

### 页面导航

- **主页**：实时时间/日期、天气信息、阅读统计、进度条
- **书架**：网格布局显示书籍（三体、百年孤独、活着、1984、红楼梦），按 Enter 进入阅读模式
- **设置**：字体/字号/行距/字间距/页边距配置

### Mock 数据

仿真使用 Mock 数据层模拟真实硬件：

| 模块 | 数据来源 | 说明 |
|------|----------|------|
| RTC 时间 | 系统真实时间 | 实时显示当前时间 |
| 天气 | 预设场景 | 北京/上海/深圳天气循环 |
| 书籍 | 5 本中文书 | 三体、百年孤独、活着、1984、红楼梦 |
| 阅读统计 | 模拟数据 | 今日 38 分钟，累计 127 小时 |

## 项目结构

```
ai-bookreader/
├── src/                    # ESP32 固件源码
│   ├── hal/                # 硬件抽象层（EPD/按键/电量/RTC/SD）
│   ├── engine/             # 引擎层（配置/事件/排版/解析）
│   ├── ui/                 # UI 层（页面状态机/组件）
│   └── main.c              # 固件入口
├── app/                    # PC 仿真
│   ├── sim_main.c          # 仿真入口
│   ├── renderer_sdl.c      # SDL2 渲染器
│   ├── shared/             # 共享页面层代码
│   └── Makefile            # 仿真构建脚本
├── platformio.ini          # PlatformIO 配置
└── requires_01.md          # 产品需求文档
```
